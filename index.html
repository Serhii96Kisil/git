Команды git

git config ­­--global user.name - используется для присвоения имени

git config --­­global user.email kisilsergej96@gmail.com - указываем имеил куда будут заливаться коммиты

git init для создания репозитория

Cd для изменение каталогов,навигация,например для перехода в другую папку

git status,посмотреть изменения ,которые применялись в репозитории

git add. Для добавления изменений

git rm —cached (название элемента)удаляет из отслеживаемых документ

git reset HEAD (название элемента),документ остается отслеживаемым но удаляет файл из включения в коммит, используется при ошибочном git add

git commit -m "комментарий к коммиту" - создание коммита и его описание

git remote add origin http… для создания записи о новом подключении к удаленному репозиторию

git push (название ветки,куда заливаем коммит) залить коммиты в репозиторий

git merge для слияния веток(добавления одной ветки к другой)

Git log -p (выводит подробные изменения в коммите)

Git diff показывает какие были изменения в документе

Git diff —staged показывает какие изменения были сделаны и уже добавлены в index через add
